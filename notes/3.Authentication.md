# Authentication & Authorization

Authentication is the process of verifying who a user is, while authorization is the process of verifying what they have access to.

To implement authentication in our Express application, we use **Passport.js**, which is a popular authentication middleware for Node.js.

---

## 1. Installation of Passport

We need to install `passport`, `passport-local` (for local username/password authentication), and `passport-local-mongoose` (an easy mongoose plugin for passport).

Run the following command in your terminal:

```bash
npm i passport
npm i passport-local
npm i passport-local-mongoose
```

For documentation:
* [Passport Local Strategy Docs](https://www.passportjs.org/packages/passport-local/)
* [Passport Local Mongoose Docs](https://github.com/saintedlama/passport-local-mongoose)

## 2. Defining the User Model

We configure a User model schema in [models/user.js](file:///c:/Users/Lenovo/OneDrive/Documents/Coding/Backend%20Apna%20College/7.Project/models/user.js). We only define a field for `email` because **`passport-local-mongoose`** automatically injects the `username`, `hash` (encrypted password), and `salt` fields for us.

```javascript
const mongoose = require("mongoose");
const Schema = mongoose.Schema;
const passportlocalMongoose = require("passport-local-mongoose").default; // Note the .default for v8/9+

const userSchema = new Schema({
    email: {
        type: String,
        required: true,
    }
});

// plugin adds username, hash, and salt fields, along with extra helper methods
userSchema.plugin(passportlocalMongoose);

module.exports = mongoose.model('User', userSchema);
```

## 3. Configuring Passport in `app.js`

To set up passport in [app.js](file:///c:/Users/Lenovo/OneDrive/Documents/Coding/Backend%20Apna%20College/7.Project/app.js):

### Step 3.1: Import Required Packages
```javascript
const passport = require("passport");   
const localStrategy = require("passport-local");
const User = require("./models/user.js");
```

### Step 3.2: Initialize Middleware
We must initialize passport **after** the express-session middleware has been defined:
```javascript
app.use(session(sessionOptions)); // Session must come first
app.use(flash());

app.use(passport.initialize()); // Initializes passport middleware
app.use(passport.session()); // Enables persistent login sessions (users don't have to log in on every request)
```

### Step 3.3: Configure Strategy and Serialization
```javascript
passport.use(new localStrategy(User.authenticate())); // Uses local strategy for login, using Mongoose model's authenticate() method
passport.serializeUser(User.serializeUser()); // Defines how the user object is stored/serialized in the session
passport.deserializeUser(User.deserializeUser()); // Defines how the user object is retrieved/deserialized from the session
```

### 5.2 User Router: [routes/user.js](file:///c:/Users/Lenovo/OneDrive/Documents/Coding/Backend%20Apna%20College/7.Project/routes/user.js)

We define route handlers for displaying forms and processing authentication logic:

```javascript
const express = require("express");
const router = express.Router({ mergeParams: true });
const User = require("../models/user.js");
const wrapAsync = require("../utils/wrapAsync.js");
const passport = require("passport");

// 1. Render Signup Form
router.get("/signup" , (req, res) => {
    res.render("users/singup.ejs");
});

// 2. Process Signup Request
router.post("/singup", wrapAsync(async (req, res) => {
    try {
        let { username, email, password } = req.body;
        const newUser = new User({ email, username });
        const registerUser = await User.register(newUser, password);
        req.flash("success", "Welcome to Wonderlust!");
        res.redirect("/listings");
    } catch(e) {
        req.flash("error", e.message); // Flash user-friendly mongoose validation errors (like "User already exists")
        res.redirect("/signup");
    }
}));

// 3. Render Login Form
router.get("/login", (req, res) => {
    res.render("users/login.ejs");
});

// 4. Process Login Request (Passport Authentication Middleware)
router.post("/login", passport.authenticate("local", {
        failureRedirect: "/login",
        failureFlash: true // Automatically flash error if strategy verification fails
    }), async (req, res) => {
        req.flash("success", "Welcome to Wonderlust!");
        res.redirect("/listings");    
});

module.exports = router;
```
