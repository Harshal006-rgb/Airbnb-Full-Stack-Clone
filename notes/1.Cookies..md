# Cookie Parser in Express

Cookies are small data files stored on the client's browser to track state.

## 1. Installation

```bash
npm install cookie-parser
```

## 2. Setup

```javascript
const cookieParser = require("cookie-parser");
app.use(cookieParser());
 // Parses Cookie header and populates req.cookies
```

## 3. Sending Cookies (Writing)

You can send cookies to the client using `res.cookie()`:

```javascript
res.cookie(name, value, [options]);
```

### Example:
```javascript
app.get("/getcookies", (req, res) => {
    res.cookie("name", "Harshal");
    res.cookie("madeIn", "India");
    res.send("sent you some cookies!");
});
```

## 4. Reading Cookies (Parsing)

Read client cookies using the `req.cookies` object:

### Example 1:
```javascript
app.get("/", (req, res) => {
    console.dir(req.cookies);
    res.send("Hi, I am root!");
});
```

### Example 2:
```javascript
app.get("/getcookies", (req, res) => {
    let { name = "anonymous" } = req.cookies;
    res.send(`Hi, ${name}`);
});
```

## 5. Signed Cookies

Signed cookies are cookies that have a signature attached to them. This signature is generated using a secret string (key) and is used to detect if the client has modified the cookie.

### Setup (With Secret):
```javascript
app.use(cookieParser("secretcode")); // Pass a secret string
```

### Sending a Signed Cookie:
Pass `{ signed: true }` in the options object of `res.cookie()`.
```javascript
app.get("/getsignedcookie", (req, res) => {
    res.cookie("color", "red", { signed: true });
    res.send("done!");
});
```

### Verifying and Reading Signed Cookies:
Signed cookies are stored in `req.signedCookies` instead of `req.cookies`. If a signed cookie has been modified on the client-side, it will not be present in `req.signedCookies` (it will be `false` or excluded).
```javascript
app.get("/verify", (req, res) => {
    res.send(req.signedCookies);
});
```

## Complete Classroom Demo Code

Here is the exact code implemented in [classroom/server.js](file:///c:/Users/Lenovo/OneDrive/Documents/Coding/Backend%20Apna%20College/7.Project/classroom/server.js):

```javascript
const express = require("express");
const app = express();
const users = require("./routes/user.js");
const posts = require("./routes/post.js");
const cookieParser = require("cookie-parser");

app.use(cookieParser("secretcode"));

app.get("/getsignedcookie", (req, res) => {
  res.cookie("color", "red", { signed: true });
  res.send("done!");
});

app.get("/verify", (req, res) => {
  res.send(req.signedCookies);
});

app.get("/getcookies", (req, res) => {
  res.cookie("name", "Harshal");
  res.cookie("madeIn", "India");
  res.send("sent you some cookies!");
});

app.get("/", (req, res) => {
  let { name } = req.cookies;
  res.send(`Hi, ${name}`);
});

app.use("/users", users);
app.use("/posts", posts);

app.listen(3000, () => {
  console.log("server is listening to port 3000");
});
```
